# 🐍 freecodecamp-python-projects

This repository is a collection of the Python projects I built while earning the **freeCodeCamp Python** certification.

---

## 🚀 Projects

### ➔ Report Card Printer
Automates the formatting and display of student data into a clean, readable report card.
* **Skills practiced:**
    * Variables & Data Types
    * Program Logic
    * Output & Debugging

### ➔ Employee Profile Generator
Parses and formats raw data strings into standardized corporate employee profiles.
* **Skills practiced:**
    * Variable Type Conversion
    * String Slicing & Manipulation

### ➔ Bill Splitter
A utility for calculating group expenses, including tip percentages and fair distribution.
* **Skills practiced:** 
    * Mathematical Operations
    * f-Strings for formatting
    * Rounding Values

### ➔ Movie Ticket Booking Calculator
A tool that calculates ticket pricing based on age, time, and membership status using logic gates.
* **Skills practiced:**
    * Comparison Operators
    * Nested Conditionals
    * Boolean Logic
  
### ➔ Weather Travel Planner
A decision-making script that determines the best mode of transport based on distance, weather conditions, and vehicle availability.
* **Skills practiced:**
    * Complex Conditional Statements (`if`, `elif`, `else`)
    * Boolean Logic (`and`, `or`, `not`)
    * Logical flow and prioritization
